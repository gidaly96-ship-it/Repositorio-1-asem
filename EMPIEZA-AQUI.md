## 🔗 **PASO 2: SUBE A GITHUB** (5 minutos)

### Opción Fácil - GitHub Desktop:

```
1. Descarga: https://desktop.github.com
2. Instala y abre GitHub Desktop
3. Inicia sesión (crea cuenta si no tienes)
4. Clic: "File" → "Add Local Repository"
5. Selecciona la carpeta de tu proyecto
6. Clic: "Create Repository"
7. Escribe nombre: "consultorio-app"
8. Clic: "Publish repository"
9. ¡Listo!
```

**¿Necesitas más ayuda?** Lee: `GUIA-DESPLIEGUE-VERCEL.md`

---

## 🚀 **PASO 3: DESPLIEGA EN VERCEL** (3 minutos)

```
1. Ve a: https://vercel.com
2. Clic: "Sign Up" (o "Log In")
3. Selecciona: "Continue with GitHub"
4. Clic: "New Project"
5. Busca tu repo: "consultorio-app"
6. Clic: "Import"
7. NO cambies nada
8. Clic: "Deploy"
9. Espera 1-2 minutos ☕
10. ¡LISTO! Copia tu URL: https://tu-app.vercel.app
```
