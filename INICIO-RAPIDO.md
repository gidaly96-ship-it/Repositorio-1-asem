## 🚀 **PASO 1: DESPLEGAR EN VERCEL (5 minutos)**

### La forma más fácil:

```
1. Exporta tu proyecto desde Figma Make
   (Busca botón "Export" o "Descargar código")

2. Ve a: https://github.com
   - Crea cuenta si no tienes
   - Clic en "New repository"
   - Nombre: "mi-app-consultorio"
   - Clic en "Create repository"

3. Descarga GitHub Desktop: https://desktop.github.com
   - Instala y abre
   - "Add Local Repository" → selecciona tu carpeta exportada
   - "Publish Repository" → confirma

4. Ve a: https://vercel.com
   - Inicia sesión con GitHub
   - "New Project"
   - Selecciona tu repositorio "mi-app-consultorio"
   - "Deploy"
   - Espera 2 minutos...
   - ¡LISTO! Copia tu URL: https://mi-app-xxx.vercel.app
```

- Recarga la página después de cambiar el nombre
